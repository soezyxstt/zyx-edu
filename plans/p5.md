## Backend spec
1. Nodes: concepts of the course (grouped KOs). Edges: `knowledge_relationships` type `prerequisite`. Toposort (Kahn). Cycle: log warning (admin warning surface or console + Sentry), break the edge whose target is later in chapter order, continue. Deterministic.
2. Ready-set ordering: lowest mastery first, then chapter order, then KO importance.
3. Step shape: `{conceptName, status: 'locked'|'available'|'in_progress'|'mastered', actions: {moduleHref?, quizTemplateId?, flashcardCount?}, estimatedMinutes}`. Status: mastered = score >= 70 AND confidence >= 50; locked = any prereq score < 40; in_progress = evidence exists, not mastered; else available. Estimate: material reading time + 10 per quiz + 0.5 per flashcard, rounded up.
4. Persist whole path JSON; recompute on mastery change (target < 200 ms for 100 concepts, pure SQL + in-memory sort).
5. No prerequisite edges at all in a course: path = chapter order, all available. Acceptable fallback; the audit makes it adaptive.

## Frontend spec
Screen: `app/courses/[id]/path/page.tsx`, tab "Study path".
- Vertical timeline, NOT cards. Container `relative pl-6`: continuous rail `absolute left-2 top-0 bottom-0 w-px bg-border`.
- Step row: `relative py-4` with node dot on the rail: `absolute -left-[1.15rem] top-5 size-3 rounded-full border-2` (perfect circle, allowed). Dot states: mastered `bg-primary border-primary`; in_progress `bg-background border-primary`; available `bg-background border-border`; locked `bg-muted border-border`.
- Row content: concept name `text-body-base font-medium` (locked: `text-muted-foreground`); right side mastery bar `w-24` + `text-body-sm text-muted-foreground tabular-nums` "{estimatedMinutes} min".
- Under the name, action links `flex gap-4 text-body-sm text-primary` (Module, Quiz, "{n} cards"); locked rows instead show `Lock size-3.5` + `text-body-sm text-muted-foreground` "unlocks after {prereq}".
- First `available` step gets a left accent: `border-l-2 border-primary -ml-[2px] pl-2` and a `Badge variant="secondary"` "next".
- Motion: steps in `<Reveal translateFrom="translate-y-4" duration="duration-500">`, one Reveal per step.
- States: skeleton 6 step rows; empty (no KOs): "Path appears once course content is processed."
- Dashboard course row: append `text-body-sm text-primary` link "Continue: {concept}" from the path head.
