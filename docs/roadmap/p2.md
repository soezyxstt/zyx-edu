## Backend spec
1. `student_streaks`: `studentId` (pk), `currentStreak` (int), `longestStreak` (int), `lastActiveDate` (text yyyy-mm-dd). Update lazily on `/api/student/today` GET: if `lastActiveDate` is today, no-op; yesterday, increment on first event today; older, reset to 1 when an event exists today, else just report. A day counts when >= 1 `learning_events` row exists for that local date.
2. `daily_recommendations`: `id`, `studentId`, `date` (text yyyy-mm-dd), `payload` (json), `completedItems` (json array of item ids), `generatedAt`. Unique `(studentId, date)`. Reuse the row all day.
3. Plan builder (priority order, all SQL):
   a. Flashcards: due count via SM-2 query, cap 20. Item `{id:'flashcards', count, href}`.
   b. Quiz: 1 published `quiz_templates` whose tags intersect the student's bottom-3 mastery concepts AND not passed (no attempt with score >= 70). Item carries template id + concept name.
   c. Module: 1 published `website_materials` chapter covering the lowest-mastery concept, not completed in `student_chapter_progress`.
   d. Fallback (no weak concepts yet): next unstarted chapter in course order.
   P5 note: P5 switches the source for (b) and (c) to the study-path head. Keep the payload shape identical.
4. `GET /api/student/today` returns `{streak: {current, longest}, items: [{id, kind, title, count?, href, done}]}`. `POST /api/student/today/complete {itemId}` plus auto-complete hooks from the underlying features.

## Frontend spec
Screen: dashboard top (edit `app/dashboard/page.tsx`).
- Region 1, greeting row: `flex items-end justify-between pb-6 border-b border-border`. Left: `h1 text-h4 font-heading` (mobile `text-h5`) "Welcome back, {firstName}". Right: `<StreakTag>`.
- `StreakTag`: `div` `flex items-center gap-1.5 rounded-md border border-border px-2.5 py-1`. `Flame size-4 text-brand-secondary` + `text-body-sm font-semibold tabular-nums` "{n} days". No background fill, no pill. Zero-streak: icon `text-muted-foreground`, text "Start today".
- Region 2, Today's plan: `mt-8`. Heading `h2 text-h6 font-heading` "Today" + right-aligned `text-body-sm text-muted-foreground` "{done}/{total} done", `border-b border-border pb-2`.
- Items: ordered checklist, `divide-y divide-border`. Row `py-3 flex items-center gap-4`:
  - Check circle: `size-5 rounded-full border` (allowed: perfect circle). Done: `bg-primary border-primary` with `Check size-3 text-primary-foreground`, `transition-colors duration-150`.
  - Kind icon: `Layers` flashcards, `ListChecks` quiz, `BookOpen` module, all `size-4 text-muted-foreground`.
  - Title `text-body-base font-medium`, done adds `line-through text-muted-foreground`.
  - Count badge when present: `Badge variant="secondary"` "{n}".
  - CTA: `Button variant="ghost" size="sm"` with `ArrowRight size-4`, links to `href`.
- Motion: rows stagger per UI-STD 2.4. Completing an item animates the check fill only (no row movement).
- Empty (new student): single row, copy "Start your first chapter", one `Button variant="outline" size="sm"`. All-done: single row `text-muted-foreground` "All done for today."
- Sidebar: same `StreakTag`, compact, above the sign-out area.
